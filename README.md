# Health Insurance Premium Price Prediction

A machine learning application that predicts health insurance premium prices based on various personal and demographic factors.

## 🌐 Live Demo

The application is hosted and available at: [https://jarvis-programmer-eqdjgsigjshpsnntwmswn3.streamlit.app/](https://health-insurence-primimum-price-pridiction-git-ee9ehkxaaeskb2y.streamlit.app/)

## 📋 Overview

This project uses machine learning algorithms to predict health insurance premium prices. By analyzing factors such as age, BMI, smoking status, region, and other relevant variables, the model provides accurate estimates of insurance premiums for individuals.

## 🎯 Features

- **Predictive Analytics**: Accurate predictions of health insurance premiums
- **User-Friendly Interface**: Interactive Streamlit web application
- **Multiple Input Parameters**: Considers various demographic and health factors
- **Real-time Predictions**: Get instant premium estimates
- **Data Visualization**: Visual insights into the prediction factors

## 🛠️ Technology Stack

- **Language**: Python 100%
- **Framework**: Streamlit (for web application)
- **Machine Learning**: scikit-learn, pandas, numpy
- **Data Analysis**: matplotlib, seaborn

## 📦 Installation

1. Clone the repository:
```bash
git clone https://github.com/Kif30/health-insurence-primimum-price-pridiction-.git
cd health-insurence-primimum-price-pridiction-
```

2. Create a virtual environment:
```bash
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
```

3. Install required dependencies:
```bash
pip install -r requirements.txt
```

## 🚀 Usage

To run the application locally:

```bash
streamlit run app.py
```

Then open your browser and navigate to `http://localhost:8501`

## 💡 How It Works

1. **Data Input**: Enter your personal information (age, BMI, smoking status, region, etc.)
2. **Model Processing**: The machine learning model processes your inputs
3. **Premium Prediction**: Receive your estimated health insurance premium price
4. **Analysis**: View factors that influence your premium

## 📊 Input Parameters

The model considers the following factors:
- **Age**: Your age in years
- **BMI**: Body Mass Index
- **Smoking Status**: Whether you smoke or not
- **Region**: Geographic region
- **Family History**: Pre-existing conditions and medical history
- **Other Demographic Factors**: Additional relevant information

## 🧠 Model Details

The prediction model is trained on historical insurance data and uses machine learning techniques to identify patterns and relationships between personal factors and premium prices.
